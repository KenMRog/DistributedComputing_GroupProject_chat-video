# ScreenShare Application Requirements

# =============================================================================
# BACKEND REQUIREMENTS (Java/Spring Boot)
# =============================================================================

# Java Development Kit
# - Java 17 or higher
# - OpenJDK 17+ or Oracle JDK 17+
# Download from: https://adoptium.net/ or https://www.oracle.com/java/

# Build Tool (Choose one)
# - Apache Maven 3.6.0 or higher
#   Download from: https://maven.apache.org/download.cgi
# - OR Gradle 8.5+ (Gradle wrapper included)

# Maven Dependencies (automatically resolved)
# - Spring Boot 3.2.0
# - Spring Web
# - Spring WebSocket
# - Spring Data JPA
# - Spring Security
# - H2 Database (development)
# - PostgreSQL Driver (production)
# - AWS SDK for SQS (2.21.29)
# - AWS SDK for SNS (2.21.29)
# - Jackson for JSON processing
# - Validation API

# =============================================================================
# FRONTEND REQUIREMENTS (React/Node.js)
# =============================================================================

# Node.js
# - Node.js 16.0.0 or higher
# - npm 7.0.0 or higher (comes with Node.js)
# Download from: https://nodejs.org/

# Node.js Dependencies (installed via npm install)
# - React 18.2.0
# - React DOM 18.2.0
# - React Scripts 5.0.1
# - Material-UI 5.14.20
# - Emotion (React styling) 11.11.1
# - Socket.io Client 4.7.4
# - SockJS Client 1.6.1
# - STOMP.js 7.0.0
# - Styled Components 6.1.1
# - Web Vitals 2.1.4

# =============================================================================
# DATABASE REQUIREMENTS
# =============================================================================

# Development Database
# - H2 Database (in-memory, included with Spring Boot)
# - Access via: http://localhost:8080/api/h2-console

# Production Database
# - PostgreSQL 12.0 or higher
# - Or MySQL 8.0 or higher
# - Or any JDBC-compatible database

# =============================================================================
# CLOUD SERVICES (Optional - for production)
# =============================================================================

# AWS Services
# - AWS SQS (Simple Queue Service)
# - AWS SNS (Simple Notification Service)
# - AWS credentials with appropriate permissions

# Alternative Message Queue Services
# - RabbitMQ
# - Apache Kafka
# - Redis Pub/Sub

# =============================================================================
# DEVELOPMENT TOOLS (Optional but recommended)
# =============================================================================

# IDE
# - IntelliJ IDEA, Eclipse, or VS Code
# - Java extension pack for VS Code

# Version Control
# - Git 2.20 or higher
# Download from: https://git-scm.com/

# Container Platform (Optional)
# - Docker 20.10 or higher
# - Docker Compose 2.0 or higher
# Download from: https://www.docker.com/

# =============================================================================
# BROWSER REQUIREMENTS
# =============================================================================

# Supported Browsers
# - Chrome 90+
# - Firefox 88+
# - Safari 14+
# - Edge 90+

# Browser Features Required
# - WebRTC support (for screen sharing)
# - WebSocket support
# - ES6+ JavaScript support

# =============================================================================
# INSTALLATION VERIFICATION
# =============================================================================

# To verify all requirements are met, run these commands:

# Check Java version
# java -version

# Check Maven version
# mvn --version

# Check Node.js version
# node --version

# Check npm version
# npm --version

# Check Git version
# git --version

# =============================================================================
# ENVIRONMENT VARIABLES (Production)
# =============================================================================

# Database Configuration
# DB_URL=jdbc:postgresql://localhost:5432/screenshare
# DB_USERNAME=your_username
# DB_PASSWORD=your_password

# AWS Configuration
# AWS_REGION=us-east-1
# AWS_ACCESS_KEY_ID=your_access_key
# AWS_SECRET_ACCESS_KEY=your_secret_key
# SQS_QUEUE_URL=your_sqs_queue_url
# SNS_TOPIC_ARN=your_sns_topic_arn

# Application Configuration
# SERVER_PORT=8080
# FRONTEND_URL=http://localhost:3000

# =============================================================================
# QUICK SETUP COMMANDS
# =============================================================================

# 1. Install Node.js dependencies
# cd frontend && npm install

# 2. Install Java dependencies (via Maven)
# cd backend && mvn clean install

# 3. Start the application
# Windows: start.bat
# Unix/Linux/Mac: ./start.sh

# 4. Or start with Docker
# docker-compose up -d
